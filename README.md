## Описание

**Clippy But Something Wrong** - это мод для Minecraft Fabric 1.21.5, который добавляет в игру культового помощника из Microsoft Office - Клиппи (Скрепыша). Но что-то явно не так...

### Основные особенности

- **Ностальгия**: НЕВЕРОЯТНАЯ ностальгия
- **Хоррор**: НЕВЕРОЯТНЫЕ страшилки
- **Сюжет**: НЕВЕРОЯТНО интересный сюжет
- **Меню**: НЕВЕРОЯТНОЕ главное меню в стиле рабочего стола Windows XP

### Тип мода

Это **клиентский мод (Client-side)**, что означает:
- ✅ Вы можете установить мод только себе и играть на любых серверах
- ✅ Не требует установки на сервере
- ⚠️ Другие игроки не увидят эффекты мода
- ⚠️ Работает только в вашем клиенте Minecraft
