> Клиппи вернулся... но с ним явно что-то не так

## Описание

**Clippy But Something Wrong** - это мод для Minecraft Fabric 1.21.5, который добавляет в игру культового помощника из Microsoft Office - Клиппи (Скрепыша). Но что-то явно не так...

### Основные возможности

- **Ностальгия**: НЕВЕРОЯТНАЯ ностальгия
- **Хоррор**: НЕВЕРОЯТНЫЕ страшилки
- **Сюжет**: НЕВЕРОЯТНО интересный сюжет

## Установка

### Требования

- **Minecraft**: 1.21.5
- **[Fabric Loader](https://fabricmc.net/use/)**: 0.16.0 или выше
- **[Fabric API](https://modrinth.com/mod/fabric-api)**: Последняя версия для 1.21.5
 
### Шаги установки

1. Установите **Fabric Loader** для Minecraft 1.21.5
2. Скачайте последнюю версию мода из раздела [Releases](../../releases)
3. Поместите скачанный `.jar` файл в папку `mods` (`.minecraft/mods`)
4. Скачайте и установите **Fabric API** (если еще не установлен)
5. Запустите игру через профиль Fabric
